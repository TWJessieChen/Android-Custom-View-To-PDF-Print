# Android-Custom-View-To-PDF-Print
Android custom view to pdf print

## 目的
此範例code，最終目的是要影印出ECG圖文報告出來，
並且格子數要符合兩小格一公分大小才行，
同時必須支援不同尺寸的平板輸出，
ECG也有區分3x4, 2x6, 1x12不同呈現方式的波形圖，
